# 🎂 Celebration Protocol v2.exe (生日快乐！)

一个充满科技感与仪式感的数字生日蛋糕交互项目。

## 🌟 项目亮点

- **代码流动画**：页面左侧实时展示 HTML/CSS/JS 的“注入”过程，模拟系统初始化。
- **动态视觉重构**：随着代码的输入，右侧实时渲染出一个精致的三层数字蛋糕。
- **互动体验**：
  - **彩色礼花**：点击页面任意位置即可触发随机颜色的粒子礼花效果。
  - **许愿仪式**：点击蜡烛火焰可以将其“吹灭”，并获得专属的许愿成功提示。
- **现代美学**：采用柔和的柠檬黄、草莓粉和香草色调，结合毛玻璃效果（Backdrop Filter）。

## 🚀 如何运行

该项目无需任何构建步骤，直接在浏览器中打开即可体验：

1. 克隆或下载本仓库。
2. 双击 `index.html` 或通过本地服务器（如 Live Server）打开。

## 🛠️ 技术实现

- **HTML5/CSS3**：使用纯 CSS 绘制蛋糕几何结构，并利用 `box-shadow` 模拟糖果装饰。
- **原生 JavaScript**：实现异步打字机效果、语法高亮渲染以及基于 Canvas 逻辑的粒子动画。
- **响应式设计**：支持不同尺寸屏幕的视觉自适应。
